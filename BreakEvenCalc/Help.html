Brek Even Calculator Help

User Type : Admin User

Enter Username ,Password,CompanyName. It will display list of records.

Select record from the list, it will show detail of record.

User Type : Non Admin User

Enter Username, password and Company Name

Once user is logged in it will display BreakEven calculation page.

In that page enter the following input fields in numeric form: Total Sale,Costof Sale, Average Sale,Conversion Rate,Time Period,JobName

User can enter fixed cost directly or can click on Expenses Button to calculate expense using expense calculator. Once Expense button clicked, it goes to the Expense Calculation Page.

In Expense calculator Page, select values for time like weekly, monthly, quarterly yearly from drop down and enter the expense in Field.
Then click on Cal total .That will calculate total expense and it will be set in cost field of previous page.

Click on BreakEven button to calculate breakeven for entered value.

To save record in db click on Save button.

Logout is provided to logout from the application
